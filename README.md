# Signature Verification System

An AI-powered signature verification system using:

- YOLO for signature detection
- Siamese Neural Network for verification
- PyTorch
- OpenCV

## Workflow

Document
→ Signature Detection
→ Signature Cropping
→ Signature Verification
→ Genuine / Forged

## Technologies

- Python
- PyTorch
- Ultralytics YOLO
- OpenCV
